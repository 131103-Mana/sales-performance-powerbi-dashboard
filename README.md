Sales Performance & Customer Insights Dashboard
📊 Project Overview

This project analyzes sales performance, customer behavior, return patterns, and discount impact from 2023 to 2025 using Excel and Power BI.

The objective of this project is to transform raw sales data into meaningful business insights that support data-driven decision-making.

🛠 Tools & Technologies Used

Microsoft Excel – Data Cleaning & Exploratory Analysis (Pivot Tables)

Power BI – Data Modeling & Interactive Dashboard Development

DAX – Calculated Measures (KPIs, Return Rate, Discount Impact, AOV)

📈 Project Workflow

Data cleaning and preprocessing in Excel

Exploratory analysis using Pivot Tables

Data import and modeling in Power BI

Creation of DAX measures for key performance metrics

Development of interactive dashboard visuals

🔍 Key Dashboard Features

KPI Summary (Total Revenue, Total Orders, Average Rating, Return Rate, AOV, High Rating %)

Monthly Category Performance Heatmap

Revenue vs Return Trend Analysis

Regional Revenue Distribution

Revenue Loss Due to Discounts (Waterfall Analysis)

Interactive Filters (Month & Payment Method)

💡 Key Insights

Certain product categories show significant revenue erosion due to discounting.

Revenue demonstrates month-over-month variations indicating possible seasonality.

Regional contribution analysis highlights high-performing markets.

Return rate trends may indicate product quality or delivery-related issues.
