# Sales Performance & Customer Insights Dashboard

## Project Overview
This project analyzes sales performance, customer behavior, return rates, and discount impact from 2023 to 2025 using Power BI.

## Tools Used
- Microsoft Excel (Data Cleaning & Pivot Analysis)
- Power BI (Data Modeling & Visualization)
- DAX (Calculated Measures)

## Key Features
- KPI Summary (Revenue, Orders, Rating, Return Rate, AOV)
- Monthly Category Performance Heatmap
- Revenue vs Return Trend Analysis
- Regional Revenue Distribution
- Discount Impact Analysis (Waterfall Chart)

## Key Insight
The dashboard highlights revenue loss due to discounts and helps identify category-level pricing strategy effectiveness.
